# spellcast

Discord SpellCast game clone\
afterwards the plan is to create an algo that finds the word that gets you the most points

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
